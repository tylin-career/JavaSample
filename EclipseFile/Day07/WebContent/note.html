九大隱式對象

不需要建立對象就可以使用虛擬名進行呼叫方法

javax.servlet.jsp.PageContext pageContext:
這個頁面的內容，也可以取其他八個隱式對象(自訂標籤)

javax.servlet.http.HttpSession session
這是瀏覽器與服務器的一次會話(HttpSession 對象)

javax.servlet.ServletContext application
代表這個專案 (ServletContext)，取的初始化參數

javax.servlet.ServletConfig config
當前的jsp頁面，如servletConfig(開發非常少用)

javax.servlet.jsp.JspWriter out
直接輸出於jsp頁面 %=相同 %>

javax.servlet.http.HttpServletRequest request

javax.servlet.http.HttpServletResponse response

java.lang.Object page = this

java.util.Exception e
須加上 isErrorPage="true"方可以使用

作用大小 pageContent(小) request session application(大) 轉發 request.getRequestDispatcher("WEB-INF/index.jsp").forward(request, response); 重定向 response.sendRedirect("http://www.yahoo.com.tw"); errorPage: 可以加在web.xml or index 上面的 errorPage Redirect
